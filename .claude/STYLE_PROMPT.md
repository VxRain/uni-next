# Uni-App 页面设计风格指南

## 核心设计原则

### 1. 层级化卡片布局
- **三层背景体系**：页面背景（浅色）→ 卡片背景（白色）→ 卡片内项目背景（略深/略浅）
- **清晰的信息分组**：每个卡片是一个独立的信息单元
- **一致的卡片间距**：卡片之间保持统一间距

### 2. 信息视觉分层
使用三个层次区分信息重要性：
- **主标题**：最大字号 + 粗体 + 最深色
- **描述/正文**：中等字号 + 常规字重 + 中性色
- **辅助信息**：小字号 + 浅色

### 3. 图标辅助设计
- **图标 + 文字组合**：统计信息、功能入口等使用图标辅助识别
- **统一图标库**：使用 Carbon 图标集
- **图标大小与场景匹配**：常规信息小图标，功能入口大图标

### 4. 轻量交互反馈
- **快速过渡**：动画时长 150-200ms
- **微妙反馈**：点击时轻微缩放或背景色变化
- **即时响应**：用户操作立即给出视觉反馈

## 标准布局模式

### 列表卡片
```
┌─────────────────────────┐
│ 标题                →  │  ← 头部：标题+箭头
│ 副标题/标签              │  ← 副标题（可选）
│ 描述文字                 │  ← 描述内容
├─────────────────────────┤  ← 分割线
│ 🔗 120 题  🏷 2个套餐   │  ← 底部信息栏
└─────────────────────────┘
```

## 交互状态规范

### 加载状态
- 旋转动画图标 + "加载中..." 文字
- 居中显示，上下内边距较大

### 错误状态
- 错误图标 + 错误信息 + 重试按钮
- 居中显示，按钮使用主题色

### 空状态
- 灰色图标 + 主提示文字 + 副提示文字
- 居中显示

### 选中状态
- 背景色变为浅色主题色
- 边框变为主题色
- 右上角显示选中图标

## 固定元素规范

### 顶部搜索栏（粘性）
- `position: sticky`
- 白色背景，圆角
- 图标 + 输入框 + 清除按钮

### 底部操作栏
- `position: fixed`
- 必须适配安全区域：`padding-bottom: calc(...) + env(safe-area-inset-bottom))`
- 上边框分隔
- 左侧信息，右侧操作按钮

## 按钮设计规范

### 主要操作按钮
- 渐变背景
- 阴影效果
- 圆角较大（接近圆形）
- 白色文字，粗体

### 次要操作按钮
- 纯色背景（主题色）
- 中等圆角
- 白色文字

## 间距节奏

建立统一的间距系统（不必固定像素值，保持相对关系）：
- 卡片内边距：中等间距
- 卡片间距：中小间距
- 模块标题下边距：中小间距
- 图标与文字：极小间距
- 行元素间距：小间距
- 模块间距：中小间距

## 关键实现要点

1. **所有可点击元素都要有交互反馈**
2. **底部固定元素必须适配安全区域**
3. **使用 flex + gap 处理横向排列**
4. **使用 flex-direction: column 处理纵向列表**
5. **图标使用统一图标库（Carbon）**
6. **状态图标大小：加载 < 错误 ≈ 空状态**
7. **卡片底部信息栏用分割线分隔**
